# Changelog

All notable changes to this project are documented in this file.

## [0.0.2] - 2026-07-08

### Changed

- Switched package entry to ESM (`390a460`)

### Fixed

- Tokenizer performance and PL/SQL block handling (`c33a5a1`)

### Docs

- Added standard Chinese JSDoc comments to core code and types (`bf1787d`)
- Added runtime tests and online demo docs (`e1e0ba4`)

## [0.0.1] - 2026-07-08

Initial release.

- Standard SQL, Couchbase N1QL, IBM DB2, and Oracle PL/SQL formatting
- TypeScript source with bundled type declarations
- ESM and UMD build artifacts
- Configurable indentation and named/indexed placeholder replacement
- VitePress documentation site and local Vite demo
- GitHub Pages docs deployment
